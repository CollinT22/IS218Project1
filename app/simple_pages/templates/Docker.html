Docker

What is a Docker Container?

A container is essentially a virtualized application that is created from a Docker Image. Docker containers can be adjusted and changed. It provides an isolated environment on where to work on an application.

What is a Docker Image?

An image is a static template that containers can be built off of. Image is a snapshot of a container and is created with the build command.

    Useful Docker Commands
  • Kill Image: docker kill imagename
  • To Force Something: Use -f
  • Remove Image: docker rmi imagepath -f
  • Run Docker On a Port 80:80 : docker run -p 80:80
  • Building Docker Container: docker compose up --build

Building a Docker Project

First create a DockerFile then use the command "docker build -t filename" . Then start the container with "docker run port:port filename" {% endblock %}
